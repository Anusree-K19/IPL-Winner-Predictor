# Hybrid IPL Match Winner Prediction (XGBoost + LSTM)

This project builds an end-to-end AI system to predict IPL match outcomes using both machine learning and deep learning techniques.

The system combines:
- XGBoost for pre-match winner prediction using structured match data
- LSTM for dynamic, live win probability prediction using sequential match data
- SHAP for model explainability and feature importance analysis

Key features:
- Advanced feature engineering (team form, head-to-head stats, venue performance)
- Comparison of multiple models (Logistic Regression, Random Forest, XGBoost, LSTM)
- Real-time style win probability prediction
- Visualisations including ROC curves, confusion matrices, and SHAP plots

Tech stack:
Python, Pandas, Scikit-learn, XGBoost, TensorFlow/Keras, SHAP, Streamlit

This project demonstrates practical experience in:
- Machine learning on tabular data
- Deep learning for time-series prediction
- Explainable AI
- End-to-end ML system design
